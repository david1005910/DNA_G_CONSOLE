## 2. 도입된 생물학적 특징 (Biological Features)

### 2.1 GC 함량 (GC Content)
- **정의**: 전체 DNA 서열 중 구아닌(G)과 사이토신(C)이 차지하는 비율.
- **계산식**: `(G + C) / (A + T + G + C)`
- **생물학적 의미**:
    - **열 안정성 (Thermal Stability)**: G-C 결합은 3중 수소 결합을 형성하여 A-T 결합(2중)보다 강합니다. GC 함량이 높은 바이러스는 고온 환경에서도 안정적인 구조를 유지할 가능성이 큽니다.
    - **유전자 밀도**: 일반적으로 유전자가 밀집된 영역(Gene-rich)은 GC 함량이 높은 경향이 있습니다.

### 2.2 K-mer 빈도 프로파일 (K-mer Frequency Profile)
- **정의**: 길이 `k`의 짧은 부분 서열(Word)들이 전체 서열에서 얼마나 자주 등장하는지의 분포. (본 모델에서는 `k=3` 사용)
- **의미**:
    - **게놈 서명 (Genomic Signature)**: 각 종(Species)이나 바이러스 군은 고유한 K-mer 분포 패턴을 가집니다. 이는 마치 지문처럼 작용하여 분류의 핵심적인 단서가 됩니다.
    - **코돈 사용 편향 (Codon Usage Bias)**: 3-mer 분석은 단백질 합성의 단위인 코돈의 사용 경향성을 간접적으로 반영합니다. 숙주 세포의 번역 시스템에 얼마나 잘 적응했는지를 나타낼 수 있습니다.

### 2.3 서열 엔트로피 (Shannon Entropy)
- **정의**: 서열 내 염기 정보의 무질서도(복잡도).
- **계산식**: `-Σ p(x) * log2(p(x))` (각 염기의 등장 확률 `p(x)` 기반)
- **생물학적 의미**:
    - **복잡도 (Complexity)**: 
        - **낮은 엔트로피**: `AAAAA` 또는 `ATATAT`와 같은 단순 반복 서열. 구조적 기능을 하거나 비부호화(Non-coding) 영역일 가능성.
        - **높은 엔트로피**: 정보량이 많은 복잡한 서열. 단백질을 코딩하는 유전자 영역일 가능성이 높음.
